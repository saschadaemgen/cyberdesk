Circuit isolation
Each window uses its own Tor circuit, so two Tor windows can't be linked to each other.

Tor mode routes that window through the Tor network and hides your IP address. It does not provide Tor Browser's anti-fingerprinting, and it cannot change the network-layer (TLS) fingerprint — so it is not equivalent to Tor Browser for anonymity. Fingerprinting hardening (tracking-resistance) is applied in every window — see below.

Tracking resistance

Fingerprinting hardening Always on, in every window — clearnet and Tor alike
On
What it does
Adds tiny, invisible per-session noise to the readback surfaces sites use to fingerprint you (canvas, WebGL, audio, text metrics), and reduces a few high-entropy values (CPU cores, memory, fonts, timezone). The noise is stable within a session but changes on the next launch, so a site cannot link one session to the next.
What stays real
Your operating system, browser identity (User-Agent), and language are left untouched and mutually consistent — no half-spoofing that would make you more identifiable.

This is tracking-resistance, not anonymity. It makes you harder to link across sites and sessions, but a low-population browser cannot hide in a crowd the way Tor Browser can with millions of identical users. For serious anti-fingerprinting anonymity, use Tor Browser.
